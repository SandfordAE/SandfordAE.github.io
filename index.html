SandfordAE.github.io

A simple UI created with Html and TailwindCSS,
Created to allow me (and you) to easily explore the latest additions and features I've added to the SandfordAE repo found at GitHub.

The purpose of the public repos are primarily to give me access to a curated list of code snippets and frameworks in order to allow me to quickly begin the development process,

But seeing as its a public facing repo I figured theres no harm in creating an interface that may help other people along in their way in their learning experiences and overall make it easier for them (and me) to get started generating code.
