1. - Get Started



Provision Git

First we should check to see if we have Git installed correctly.

Terminal should return you something that
looks like this including a version number.

┌──(User㉿DellKali)-[~]
└─$ git -v
git version 2.39.2

If not we can simply install it using this command.

With Git installed we are ready to initialize a folder.

Change directory to the folder you wish to use.

┌──(User㉿DellKali)-[~]
└─$ cd /home/User/dev/websites/docs/pages/

Use the "git init" command to create a .git file
containing our configurations and a .gitignore
file to list files and folders we DO NOT wish
to share with the repository.

Add the folders to be tracked.

Check the status of the tracker.

Attribute this data to the Master Branch.

Give this Commit a name.

Check the log.

We should see something like this.

└─$ git log

5. - Close the lid uninterrupted



In order to close the lid without the machine powering down
we'll need to configure our systemd file.

Scroll down to find these values and edit accordingly.


EDIT ---> #HandleSuspendKey=suspend ---- TO ---> HandleSuspendKey=ignore
EDIT ---> #HandleLidSwitch=suspend ---- TO ---> HandleLidSwitch=ignore
EDIT ---> #HandleLidSwitchDocked=ignore ---- TO ---> HandleLidSwitchDocked=ignore



Once thats done we can restart our systemd.
