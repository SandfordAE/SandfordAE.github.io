5. - Close the lid uninterrupted



In order to close the lid without the machine powering down
we'll need to configure our systemd file.

Scroll down to find these values and edit accordingly.


EDIT ---> #HandleSuspendKey=suspend ---- TO ---> HandleSuspendKey=ignore
EDIT ---> #HandleLidSwitch=suspend ---- TO ---> HandleLidSwitch=ignore
EDIT ---> #HandleLidSwitchDocked=ignore ---- TO ---> HandleLidSwitchDocked=ignore



Once thats done we can restart our systemd.

7. - Fail2Ban



Provision

Simply use apt to install fail2ban

Check to see if the service is healthy.

Configure

Change directory to /etc/fail2ban.

List files in folder.

Copy the conf file to a local file.

Use nano to edit the new file.

Scroll down to find "#JAILS #SSH servers #[sshd]"
Above where it states "port = ssh" add the following.

enabled = true
bantime = 10m
maxretry = 5
findtime = 10m


Enable and Check
